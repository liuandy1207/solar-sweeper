# Solar Sweeper

Solar Sweeper is a mechatronics-based approach to cleaning solar panels involving a dual-axis cable-driven gantry system.

## Project Structure
```
solar-sweeper/
├── code.py           # 
├── keyboard_controls.py         # TMC2209 driver config and stepper movement functions
├── controller.py     # WASD keyboard input handling via pynput
├── config.py         # Pin assignments, speed settings, and constants
└── README.md
```
